# Automated File Manager

A Python automation project that organizes files in a folder based on file types
and generates simple reports.

## Features
- Automatically sorts files (PDF, images, data, etc.)
- Creates folders if they do not exist
- Keeps the directory clean and organized
- Simple and beginner-friendly code

## Technologies Used
- Python
- os module
- shutil module

## How to Run
1. Clone the repository
2. Open the project folder
3. Run:
   python main.py

## Output
Files will be moved into folders like:
- pdf/
- images/
- data/
